سیستم مدیریت بازخورد (Feedback Management System)


این پروژه یک سیستم مدیریت بازخورد تحت وب است که با استفاده از FastAPI، SQLAlchemy، SQLite، Jinja2 و Bootstrap توسعه داده شده است.

کاربران می‌توانند بازخوردهای خود را ثبت کرده، وضعیت آن‌ها را مشاهده کنند و پاسخ ادمین را دریافت نمایند. همچنین یک پنل مدیریت برای بررسی، دسته‌بندی، پاسخ‌دهی و مدیریت بازخوردها در نظر گرفته شده است.

خروج از پنل مدیریت


راهنمای اجرای پروژه در سیستم لوکال

1. دریافت پروژه

git clone <repository-url> cd feedback_system 

2. ساخت محیط مجازی

python -m venv venv 

3. فعال‌سازی محیط مجازی

در ویندوز:

venv\Scripts\activate 

در لینوکس یا مک:

source venv/bin/activate 

4. نصب وابستگی‌ها

در صورتی که فایل requirements.txt وجود داشته باشد:

pip install -r requirements.txt 

در غیر این صورت:

pip install fastapi uvicorn sqlalchemy jinja2 python-multipart itsdangerous 

5. اجرای پروژه

uvicorn main:app --reload 

6. باز کردن پروژه در مرورگر

http://127.0.0.1:8000 

توضیح تصمیم‌های فنی

اضافه کردن ویژگی دیدن بازخورد ها به علت اینکه هر بازخوردی که بزاریم دیدن نتیجش برای کاربر مهمه
در پنل مدیریت اضافه شدن جواب که مدیر در صورت نیاز میتواند جواب را برای کاربر ارسال کند و کاربر را بهتر مطلع کند از شرایط
دریافت کد تایید ایمیل که نیاز نباشد قبل از ارسال فیدبک کاربر لاگ این کرده باشه

چرا FastAPI؟

FastAPI به دلیل سرعت بالا، سادگی توسعه، اعتبارسنجی خودکار داده‌ها و ساختار مناسب برای APIهای مدرن انتخاب شد.

چرا SQLite؟

برای این پروژه از SQLite استفاده شد زیرا:

چرا از Session استفاده شد؟

برای جلوگیری از دسترسی غیرمجاز به اطلاعات کاربران از Session استفاده شده است.

مزایا:

هر کاربر فقط به بازخوردهای خود دسترسی دارد.

امکان دسترسی مستقیم به اطلاعات دیگران از طریق URL وجود ندارد.

پس از خروج کاربر دسترسی‌ها حذف می‌شوند.
